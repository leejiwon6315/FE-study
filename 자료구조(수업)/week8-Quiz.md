### 1. 다음 트리에 대한 중위 순회 결과는?
1. A B D C E F
2. A B C D E F
3. D B E C F A
4. D B A E C F
> 4번, 중위 순회는 왼쪽 노드 -> 루트 -> 오른쪽 노드 순서

<br/>

### 2. 다음 트리를 전위 순회로 운행할 경우 다섯 번째로 탐색되는 것은?
1. C
2. E
3. G
4. H
> 2번, 루트 -> 왼쪽 노드 -> 오른쪽 노드

<br/>

### 3. 그림과 같은 이진트리를 후위 순회한 결과는?
1. + * * / A B C D E
2. A / B * C * D + E
3. + * A B / * C D E
4. A B / C * D * E +
> 4번, 후위 순회는 자식 노드에서부터 탐색 

<br/>

### 4. 다음 트리에서 단말 노드 수는?

2. 3

4. 8
