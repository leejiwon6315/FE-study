```
[정점 A] -> [B] -> [C] -> [D](NULL)
[정점 B] -> [A] -> [E] -> [F](NULL)
[정점 C] -> [A] -> [F](NULL)
[정점 D] -> [A] -> [F](NULL)

[정점 F] -> [B] -> [C] -> [D] -> [G]
```

## C11
### 연결된 그래프 G의 간선들 중에서 그 간선을 제거하면 연결이 끊어지는 간선 (u, v)를 브리지(bridge)라고 한다. 주어진 그래프에서 브리지의 전체 개수를 찾아내는 멤버함수를 작성하라.
```
int findBridge(){
    int i, j;
    int count = 0;
    
    g->resetVisited();

    for(i = 0; i < g->getSize(); i++){
        for(j = 0; j < i; j++){
            if(g->getEdge(i,j) != 0){
            g->setEdge(i, j, 0);
            g->setEdge(j, i, 0);

            g->resetVisited();
				        
                if(g->findConnectedComponent()>1){
                    printf(" Bridge%d: %c-%c\n", count+1, g->getVertex(i), g->getVertex(j));
                    count++;
                }
				        
                g->setEdge(i, j, 1);
                g->setEdge(j, i, 1);
            }
        }
    }
    return count;
}
```
