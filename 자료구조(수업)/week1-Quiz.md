### 6. 다음 알고리즘의 시간 복잡도를 n에 대한 함수로 나타내고, 빅오 표기법으로도 나타내라.
```
int algorithm(int n){
  int k=0;      // 대입 연산 1번
  while(n>1){    
    n = n/2;    // 대입 연산 log2n번
    k++;        // 증가 연산 log2n번
  }
  return k;     // 반환 연산 1번
}
```
> n에 대한 함수 : 2log2n + 2 <br/>
> 시간 복잡도 : O(log2n)
